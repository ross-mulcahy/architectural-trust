# Editor Guide

## How This Theme Is Meant To Be Used

This theme is now structured for CMS users first:

- templates provide the outer shell
- page content is edited directly in the block editor
- reusable sections live in the pattern library
- editorial/news sections pull from posts automatically

You should not need to edit templates to update normal page content.

## Best Pages To Edit Directly

- `Home`
- `Products`
- `News & Intelligence`
- `Login`
- `Account Dashboard`
- `Transfers & Bill Pay`

Each of these pages now renders `post-content`, so marketers can update copy, imagery, CTAs, and section order in the editor.

## Global Areas

Update these in the Site Editor:

- Header
- Footer
- Minimal Header
- Minimal Footer

Use the Site Editor for navigation, footer links, social links, and any sitewide disclosure text.

## Pattern Library

The pattern library is designed in three layers:

### Landing / marketing sections

- `Landing Hero`
- `The Distinction`
- `Featured Products`
- `Market Intelligence`

### Modular page-builder sections

- `Advisor Callout`
- `Client FAQ`
- `Newsletter Signup`
- `Rates Comparison`
- `Support Contact Panel`
- `Trust Signals`

### Page starters

- `Starter - Campaign Landing Page`
- `Starter - Product Detail Page`

Use the starter patterns to create a new page quickly, then swap, remove, or reorder sections as needed.

## Editorial Workflow

- publish posts normally in WordPress
- the homepage intelligence section updates automatically
- the News landing page updates automatically
- related-article sections on single posts update automatically

If a marketer needs to change a headline card in those areas, the right place is usually the post itself, not the page template.

## Good Editing Habits

- Keep homepage and campaign sections as regular page content, not synced patterns.
- Reserve synced/global patterns for truly repeated content like disclosures or shared sign-up strips.
- Avoid editing the raw template files unless you are changing sitewide structure.
- Review demo/legal placeholder copy before any public-facing handoff.
